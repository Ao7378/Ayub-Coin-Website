      Proyectos de NEM

      Apostille

      Apostille es un sistema notarial integral blockchain. En este sistema, las apostillas no llevan un timestamp permanente y estático, sino que ahora estos valores pueden ser dinámicos, cambiantes y actualizables en la blockchain.

      Read more

      Autenticador NEM

      El Autenticador NEM es una aplicación móvil que proporciona una Confirmación de 2 factores para asegurar tus activos y evitar que estos puedan ser robados.

      Read more

      Herramientas NEM

      Conjunto de herramientas NEM para ayudar al usuario en las tareas básicas

      Read more

      Hot/Cold Wallet de NEM

      Hot/Cold Wallet de NEM: El primer monedero frío de NEM, sin conexión, que se puede utilizar para enviar y recibir fondos.

      Read more

      ioNEM

      ioNEM combina IOT (Internet of Things) con la blockchain.

      Read more

      Landstead

      Landstead utiliza la tecnología blockchain de NEM para crear registros de propiedad, permitiendo al gobierno y a los ciudadanos crear de forma conjunta un sistema blockchain abierto en el que se pueda confiar y pueda ser consultado por las partes interesadas.

      Read more

      NEMPay

      NEMPay es una aplicación para transferir activos, de forma segura, entre dispositivos móviles utilizando la blockchain de NEM.

      Read more

      NEMventory

      Puedes crear una aplicación SAAS segura utilizando la blockchain de NEM.

      Read more

      PacNEM

      PacNEM es un juego multijugador de navegador Pacman de código abierto. Es una muestra de lo fácil que es integrar las características de la blockchain en juegos o aplicaciones.

      Read more

      Proyecto NEM ICO

      NEM tiene como objetivo un proyecto de trabajo colaborativo que muestra cómo crear una ICO con la blockchain de NEM.

      Read more
