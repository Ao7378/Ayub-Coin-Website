      PROGRAMA DE DESARROLLO

      Programa de Recompensas para Desarrolladores

      El programa de recompensas para desarrolladores de NEM ofrece recompensas para todas las competencias, desde el desarrollador más experimentado hasta el aspirante a estudiante, hay XEMs para ganar.

      Nuestras recompensas van desde la detección de errores y problemas hasta la creación de aplicaciones encima NEM.

      Puedes conseguir la recompensa por detección de errores y problemas siguiendo los pasos que se especifican a continuación. Para la recompensa de aplicaciones a gran escala o módulos, rellena el formulario de contacto al
      final de la página.

      Recompensas por Desarrollo de Proyectos

      Recompensas: de $500 a $10,000en XEMs

      Requisitos

      • Una demo en funcionamiento escrita en ES6 Javascript ejecutándose en nodejs (lenguaje de preferencia). Dependiendo de la dimensión del proyecto y previa discusión, los proyectos también se pueden escribir en Python, C++, C#, PHP, Java y GO.
      • Código abierto en Github bajo una licencia MIT (Massachusetts Institute of Technology)
      • Un blog detallado del “cómo” se construyó el proyecto que incluya a otros en cómo construirlo así como un tutorial para usuarios otorga una mayor recompensa.

      Recompensas por Detección de Errores

      Recompensas por Errores en el Nanowallet

      Recompensas: de $100 o más, en XEMs

      Requisitos

      • Para obtener una recompensa por detección de errores simplemente debes publicar en Github el problema detalladamente y debes incluir la dirección del pago. El error no puede ser el duplicado de un error que ya haya sido reportado.
      • Adicionalmente se pagará un monto por una solicitud de extracción que se acepta una vez por cada error.

      Recompensas por Errores en el NIS

      Recompensas: de $500 a $20,000en XEMs

      Requisitos

      • Los errores se catalogan según la relevancia y severidad, de un error menor a uno crítico.
      • Los desarrolladores de NEM han probado el NIS de manera exhaustiva, así que si se detecta un fallo crítico la recompensa será muy generosa.

      Recompensas por errores en Android & iOS

      Recompensas: de $100 o más, en XEMs

      Requisitos

      • Si tienes un problema, intenta hacer una descripción detallada: el informe de tu dispositivo, la versión del sistema operativo y la versión de la aplicación. Realiza capturas de pantalla si eso ayuda a la comprensión del problema. Para recibir los XEMs debes de informar del problema en Github acompañado de tu dirección XEM.
      • Adicionalmente se pagará un monto por una solicitud de extracción que se acepta una vez por cada error.

      Formulario de Registro

      La sugerencia de nuevos temas son siempre bienvenidas en nuestra comunidad, rellena este formulario con tus sugerencias para que podamos considerarlas.

      Si deseas contribuir en nuestro proyecto, por favor, rellena este formulario y nos pondremos en contacto contigo*.

      Los campos marcados con * son obligatorios

      *Te recomendamos que no empieces a trabajar sin contactar con nosotros previamente.
