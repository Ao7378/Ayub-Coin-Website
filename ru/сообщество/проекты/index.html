      ПРОЕКТЫ СООБЩЕСТВА

      NEM

      NEM Проекты

      Все проекты NEM, перечисленные на этой странице, представляют собой проекты с открытым исходным кодом, созданные членами сообщества NEM

      ioNEM

      ioNEM объединяет Интернет вещей с помощью блокчейна.

      Read more

      Landstead

      Landstead (реестр земли) использует технологию NEM блокчейн для создания системы регистрации земли и собственности, которая позволяет правительствам и гражданам совместно создавать открытую блокчейн систему, через которую консультируются доверяют и связанные стороны.

      Read more

      NEM Authenticator

      NEM Authenticator - это мобильное приложение, которое обеспечивает 2-факторное подтверждение для защиты, учетной записи от кражи ваших средств.

      Read more

      NEM DNS

      NEM DNS демонстрирует, как можно использовать NEM блокчейн для интеллектуальной, децентрализованной и и надежного сервиса DNS

      Read more

      NEM Faucet

      Вы можете безвозмездно получить бесплатную поддержку NEM., а так же можете получить небольшое количество XEM, с помощью сервиса Faucet один раз каждые четыре часа. Для доступа к Faucet вам нужен только NEM Address в Nanowallet.

      Read more

      NEM ICO Project

      NEM Initial Coin Offering (ICO) представляет собой совместный проект, который показывает рабочий проект «Как создать ICO с NEM Blockchain».

      Read more

      NEM ID

      NEM id, безопасный способ входа в современную эпоху. Используя защищенную криптографию с открытым ключом, NEMid может регистрировать вас на совместимых веб-сайтах и в приложениях только с учетной записью NEM через отдельное приложение

      Read more

      NEM Key Recovery Service

      KRS служба восстановления ключей NEM и она отлично работает, когда пользователь хочет использовать сторонний сервис (например, кошелек), но хочет убедиться, что он может получить доступ к своим средствам при отключении службы.

      Read more

      NEM Microwallet

      NEM Microwallet - это расширение Chrome, которое позволяет пользователю быстро и доступно использовать некоторые функции NEM

      Read more

      NEM Pay

      NEMPay приложение для перевода XEM простым способом, непосредственно с мобильного устройства, с использованием NEM Blockchain

      Read more

      NEM PayContent.JS

      Идея заключается в том, чтобы легко внедрить микроплатежи для всех создателей веб-контента. Плагин представляет собой легкий javascript и попросит пользователей веб-сайта заплатить небольшую единовременную оплату за просмотр содержимого страницы. В этом случае можно просматривать статьи, блоги, изображения, страницы загрузки и т. Д.

      Read more

      NEM Tools

      NEM Tools набор инструментов NEM, чтобы помочь пользователю в выполнении основных задач.

      Read more

      NEM Vote

      Платформа для голосования NEM - это новый модуль NanoWallet, который позволяет кому-либо создавать и голосовать и данные опросы, хранятся в блокчейне NEM.

      Read more

      NEMp3

      NEMp3 - это веб-приложение для оплаты платежей, использующее API NEM

      Read more

      NEMventory

      Создайте безопасное приложение SAAS, используя Блокчейн NEM

      Read more

      PacNEM

      PacNEM - это многопользовательская игра Pacman с открытым исходным кодом. Это демонстрация того, насколько легко интегрировать функции блокчейн в игры или приложения.

      Read more

      XEMsign

      XEMsign - это мультиподписной клиент для подписи транзакций, который можно использовать в сети NEM.

      Read more

      Горячий / холодный кошелек NEM

      Горячий / холодный кошелек NEM: первый автономный кошелек, который можно использовать для отправки и получения средств.

      Read more

      Плагин для оплаты Woocommerce

      Обладая небольшими знанием о WordPress и Woocommerce, мы смогли создать распространяемый плагин WordPress, который позволяет любому магазину по всему миру безопасно принимать XEM в своих интернет-магазинах.

      Read more

      Предложения, финансируемые сообществом

      CopyrightBank

      CopyrightBank - это проект, предложенный сообществом NEM, целью которого является регистрация и проверка авторских прав на различные категории цифровых категорий, полностью оснащенные NEM с прочной бизнес-моделью, ориентированной на потребителей, компании и правительства.

      Read more

      LuxTag

      Решение LuxTag является решением для проверки подлинности в нотариальном порядке, чтобы предотвратить подделку (доказательство подлинности), кражу (доказательство права собственности) и предлагать большие данные брендам, которые реализованы на блокчейне.

      Read more

      NEM Library

      Это библиотека NEM где вы можете для себя узнать что то новое

      Read more

      Апостиль

      Апостиль - это целостная система нотариального заверения документов. В этой системе нотариального заверения документов нет постоянной и статической метки времени, и поэтому эти значения могут быть динамическими, изменчивыми и обновляемыми в блокчейне.

      Read more

      Предложения сообщества

      Фонд сообщества DAO был создан для содействия развитию экосистемы NEM путем финансирования начинающих компаний NEM, чтобы помочь начать бизнес. Он открыт для всех.

      Здесь вы можете проверить минимальные требования, чтобы приложение рассматривалось.
