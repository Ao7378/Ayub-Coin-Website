      Woocommerce支付插件-Woocommerce payment plugin

      Woocommerce支付插件-Woocommerce payment plugin

      网站

      Github 教程 Demo

       

      据BuiltWith报道，2017年4月，共有1,986,324个网站使用WooCommerce，其中包括全球最大的一百万个网站中的23,724个。这是所有网站的0.5％，或者前一百万网站的2.8％。

      NEM的优势之一就是它的API。 网络中的每个节点本身就是一个API服务器。 你可以通过简单的HTTP GET请求与区块链进行交互。这是我们在这个插件中使用的。

      由于对Wordpress和WooCommerce知之甚少，我们可以创建一个可分发的Wordpress插件，让世界各地的任何商店在他们的商店中安全地接收XEM。
      将所有面向客户的兑换转变为XEM或保留两者。你可以以美元或欧元列出你的产品，并用XEM付款。这对于以另一种货币采购商品的商店来说非常方便。汇率在需要的时候被缓存最多一分钟。支付流程建立在标准的WooCommerce流程之上，所有其他插件都应该同时这样运作。
