      NEM密钥恢复服务-NEM Key Recovery Service

      NEM密钥恢复服务-NEM Key Recovery Service

      Github Email

      在所有加密货币中，密钥的恢复是一个巨大的挑战。许多人使用加密货币的主要担忧之一是他们的资金安全。对大多数人来说这不是一件容易的事情，因为我们习惯了集中的组织来关心安全，并且在忘记密码的时候支持我们。在加密领域，如果我们遗失了密钥，我们就会永远失去我们的资金，或者在某些情况下，第三方服务会欺骗盗窃他人的加密硬币。这就是密钥恢复服务（KRS）与类似XEM的加密货币高度相关的原因。

      KRS围绕三个使用场景来构建：

      *创建新的KRS：创建新的私钥，如果丢失可以恢复。

      *加密现有的KRS：为现有的密钥恢复服务。

      * Multisig KRS：用户想要使用第三方服务（例如钱包），但是想要确保在服务关闭时他可以访问他的资金，他忘记了他的密码，或者他对该服务失去信心。
