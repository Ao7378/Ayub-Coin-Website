      NEM 项目

      NEM 项目

      此页面上列出的所有NEM项目都是由NEM社区成员构建的开源项目。

      NEM ICO项目-NEM ICO Project

      NEM首次代币发行（ICO）示例项目旨在成就合作项目，并展示如何使用NEM区块链创建ICO。

      Read more

      NEM 工具-NEM Tools

      一系列NEM工具帮助用户执行日常工作

      Read more

      NEM 微钱包-NEM Microwallet

      NEM 微钱包是一个Chrome扩展程序，允许用户以一种快速和可接入的方式获得一些NEM功能。

      Read more

      NEM 投票-NEM Vote

      NEM投票平台是一个NanoWallet的新模块，允许任何人创建存储在NEM区块链上的投票。

      Read more

      NEM 支付内容.JS-NEM PayContent.JS

      这是为所有网页内容创建者提供轻松实现微支付的方法。该插件是轻量级的JavaScript，并会要求网站的用户支付一小笔一次性支付查看页面内容。可以用来查看文章，博客，图片，下载页面等。

      Read more

      NEM 水龙头-NEM Faucet

      你可以免费获得主网（Mainnet）流通的XEM。你可以每四小时转一次“水龙头”，获得一小笔XEM。要访问水龙头，只需要Nano钱包的NEM地址。

      Read more

      NEM 认证-NEM Authenticator

      NEM身份验证是一个提供双重确认系统的移动应用程序，确保你的帐户资产在使用NEM区块链功能时不会被盗。

      Read more

      NEM 身份-NEM ID

      认识NEMid（NEM身份），这是现代登录的一种安全方式。使用安全的公钥加密技术，NEMid可以通过独立的应用程序仅通过NEM账户登录到所有兼容的网站和应用程序。

      Read more

      NEMp3-NEMp3

      NEMp3是一个下载付款的Web 应用程序，利用NEM API结合唯一的客户ID进行购买，发布到NEM区块链。

      Read more

      NEMventory-NEMventory

      使用NEM区块链创建一个安全的SAAS应用程序。

      Read more

      NEM密钥恢复服务-NEM Key Recovery Service

      如果用户想要使用第三方服务（例如钱包），但是想要确保在服务关闭时他可以访问他的资金，他忘记了他的密码，或者他对该服务失去信心，KRS是一个绝佳的选择。

      Read more

      NEM支付-NEM Pay

      NEMPay是一个使用NEM区块链以简单的方式直接从移动设备传输资产的应用程序。

      Read more

      PacNEM-PacNEM

      NEM是一个开源的多人Pacman网页游戏。这是一个简单地将区块链功能整合到游戏或应用程序中的展示。

      Read more

      Woocommerce支付插件-Woocommerce payment plugin

      由于对Wordpress和Woocommerce知之甚少，我们可以创建一个可分发的Wordpress插件，让世界各地的任何商店在他们的商店中安全地接收XEM。

      Read more

      XEM签名-XEMsign

      XEMsign（XEM签名）是一个在NEM网络上使用的多重签名的交易客户端。

      Read more

      地契-Landstead

      Landstead（地契）使用NEM区块链技术创建了一个土地和财产登记系统，使政府和公民可以共同创建一个开放式的区块链系统，受到相关方的信任和咨询。

      Read more

      热/冷钱包-Hot/Cold Wallet

      NEM热/冷钱包：你可以用来发送和接收资金的第一个NEM离线冷存储钱包

      Read more

      物联NEM-ioNEM

      物联NEM（ioNEM）将物联网与区块链相结合。

      Read more

      社区资助建议

      LuxTag

      LuxTag是一种现实生活的认证解决方案，可以防止伪造品（真伪证明），盗窃（所有权证明），并为部署在区块链的品牌提供大数据。

      Read more

      NEM库-NEM Library

      NEM库是NEM区块链的一个抽象概念，使用灵活方法创建区块链应用程序。

      Read more

      公证-Apostille

      Apostille（公证）是一个全面的区块链公证系统。在这个系统中，公证不是一次性的静态时间戳，而是区块链上动态的，移动的，可更改可更新的值。

      Read more

      版权银行-Copyright Bank

      版权银行是由NEM社区提出的一个项目，其目的是注册和验证各类数字作品的版权，完全由NEM提供支持，为消费者，公司和政府提供坚实的商业模式。

      Read more

      社区建议

      社区基金DAO的成立是为了促进NEM生态系统的发展，为NEM创业公司提供资金以帮助他们开展业务。它是对所有人开放的。
