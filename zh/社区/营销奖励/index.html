      营销计划

      营销奖励计划

      NEM团队积极鼓励社区成为NEM营销市场。
      我们欢迎和奖励优秀作家和艺术家参与我们的奖励计划。

      查阅我们的开发奖励
      开发奖励

      NEM 101 聚会

      提供此赏金是为了提高对NEM区块链的认识。如果你能够在你所在的城市举办NEM101演讲，你可能会收到以下奖励：

      • 可获得包含食品，饮料和场地在内的最高等值200美元的XEMs（要求索取收据，鼓励你寻找外部赞助）
      • 活动的组织者共享等值200美元的XEMs
      • 演示NEM101的演讲者可获得等值100美元的XEMs（组织者和演讲者可以是同一个人）

      赏金可以按照以下条款和条件提供：

        1. 组织者通过Meetup 平台组织活动
        2. 每个城市每月最多举行一次NEM101活动
        3. 关于如何组织活动和展示NEM101的材料将被发送给主办方与演讲者进行协调
        4. 将活动建议的日期，时间和地点发送至[email protected]和[email protected]。事件必须是发起建议的30天内。并非所有提案都将被接受。这将取决于你是否在NEM社区活跃，是否有其他组织者已经举行类似活动和其他一些因素
        5. 演讲必须由至少二十（20）人参加
        6. 一旦提交报告，以出席者的签名名单和合影的形式出示证明，奖金将被支付
        7. 赏金小组有最终解释权

      点击 这里 提交你的赏金声明

      发表文章奖励

      提供此奖金是为了增加媒体中NEM区块链的曝光度。奖金适用于以下情况：

      • 在自媒体上发表文章，可获得150美元
      • 在主流媒体上发表文章，可获得250美元

      赏金可以按照以下条款和条件提供：

      1. 赏金小组将确定该文章是否具有足够水平来领取这种奖励。作者可以在发布之前提交评估文章，以确认是否符合这个奖励条件
      2. 主流媒体指至少是国家级的任何在线或印刷媒体
      3. 一旦文章发表，就将立即支付奖金
      4. 每个媒体平台只分配1次奖金
      5. 赏金小组有最终解释权
      点击 这里 提交你的赏金声明

      Youtube 视频

      此奖励给予那些已经发布了促进NEM传播的视频。以下奖金适用于：

      • 一个合格的视频可获得50美元
      • 一个制作精良的视频最多可获得250美元

      赏金可以按照以下条款和条件提供：

      1. 赏金小组判断视频的质量来分配相应的赏金
      2. 视频观看超过500次后，奖金将被支付
      3. 赏金小组有最终解释权

      点击 这里 提交你的赏金声明

      NEM相关的博客文章

      此奖励给予那些为NEM写博客的人。等值100到1,000美元的XEMs奖金将支付给发表的博客文章。

      赏金可以按照以下条款和条件提供：

      1. 最好是在Markdown中撰写博客，这是在NEM论坛中撰写帖子使用的语言，方便易用，Dillinger.io是一个非常好的Markdown预览网站，请确保图片/图表在第三方网站上具有有效的链接。这里可以找到相应的指导
      2. 博客文章应保存在一个谷歌文档并提供相应链接进行评估
      3. 博客可以写成讨论主题或“如何做”的教程，可以参考这里的示例文章
      4. 一旦文章发表，就将立即支付奖金
      5. 赏金小组有最终解释权

      点击 这里 提交你的赏金声明

      商业社区延展赏金

      提供此奖励是为了提高商业界对NEM区块链的认识。如果你能够与你所在城市的主要商业组织，协会或商会组织一场与NEM的对话。

      • 奖金：100美元

      根据以下条款和条件，前50名可获得赏金：

      1. 你的城市必须是你所在国家十大城市之一。 如果你能够提供你的城市具有重要意义的原因，则可以例外
      2. 商业组织，协会或商会必须是至少有500名成员的国家级或全国性的机构
      3. 对话必须至少45分钟，并且必须由NEM基金会批准的人员进行
      4. 对话必须有至少30人参加
      5. 提交报告并附上出席名单的签名和合影的形式作为出席的证明，奖金将被立即支付
      6. 每个城市只分配1次奖金，每个国家最多分配5次奖金
      7. 赏金小组有最终解释权

      为了获得这笔奖金，请在填写 预批准表格 之前，先获得预先批准。

      如果你没有得到批准，从而不符合赏金的资格，我们将不承担任何责任。

      点击 这里 提交你的赏金声明

      交易所列表赏金

      提供此奖励是为了增加NEM交易的交易所的数量。如果你能说服交易所在其交易平台上添加NEM，你将获得等值500美元的XEMs作为奖金。

      根据以下条款和条件，前十名可获得赏金：

      1. 交易所必须运行至少6个月
      2. 交易所每天至少要交易100万美元
      3. 必须发布新闻稿，宣布将NEM添加到交易所
      4. 一旦提供NEM加入到交易所的证明，即可获得赏金
      5. 每个国家只分配1次赏金（根据交易所所在地）
      6. 赏金小组有最终解释权

      点击这里 提交你的赏金声明

      全球媒体特色赏金

      此奖励是为了增加NEM区块链在国际媒体上的曝光率。如果你能够安排一位NEM基金会成员在全球媒体平台上进行采访或专访，你将获得以下奖励：

      • 印刷媒体：奖励等值200美元的XEMs 
      • 在线媒体：奖励等值300美元的XEMs
      • 电视媒体：奖励等值500美元的XEMs

      基于以下条款和条件，上述的每种媒体将各有10次的赏金机会：

      1. 媒体平台必须具有全球影响力。印刷媒体的一个例子是时代杂志；在线媒体的一个例子是WIRED杂志；电视媒体的一个例子是CNN
      2. 一旦媒体文章发布或节目播出，奖金将被支付
      3. 每个媒体平台只分配1次赏金
      4. 赏金小组有最终解释权

      点击 这里 提交你的赏金声明

      ICO 赏金

      提供此奖励是为了鼓励使用NEM马赛克为ICO创建代币。如果你能说服一家ICO采用NEM平台发行代币，你将获得等值1,000美金的XEMs。

      根据以下条款和条件，前十名可获得赏金：

      1. 为ICO创建的代币只能是NEM马赛克
      2. 此ICO必须接受来自贡献者的XEM
      3. 必须有一个NEM的链接放在ICO网站上
      4. NEM必须在白皮书中引用
      5. 一旦ICO启动，奖金将被支付
      6. 每人只能分配1次赏金
      7. 赏金小组有最终解释权

      点击 这里 提交你的赏金声明

      学术赏金

      提供此奖励是为了鼓励对NEM区块链进行严肃的学术研究。学术研究人员进行研究，撰写和出版一篇以NEM区块链为特色的学术论文，将获得等值5,000美元的XEM的奖金。

      根据以下条款和条件，前四名可获得赏金：

      1. 研究人员必须具有获得认可的大学的博士学位
      2. 该研究项目必须得到认可的大学的批准和认可
      3. 学术论文必须在认可的期刊上发表
      4. 一旦提供以下资料，将支付1,000美元的奖金：
        • 研究提案
        • 研究人员的资质证明
        • 大学的资质证明
      5. 剩下的4,000美元将在学术论文发表后并提供以下资料后支付：
        • 载有已发表论文的期刊副本
        • 期刊认证的证明
      6. 每个学术机构只分配1次赏金
      7. 赏金小组有最终解释权

      点击 这里 提交你的赏金声明

      市场应用赏金

      提供此奖励是为了支持在NEM区块链上构建的现实使用案例的商业化。如果你成功推出了NEM驱动的应用程序，并已开始创收，将获得等值10,000美元的XEM的奖金。

      如果满足以下条款和条件，将总共提供5次赏金：

      1. 这个奖金是在盈利的基础上支付的。当从应用程序提供10,000美元的收入证明时，将支付全额奖金
      2. 每个应用程序只分配1次赏金
      3. 赏金小组有最终解释权

      点击 这里 提交你的赏金声明

      政府项目赏金

      此奖励是为了鼓励政府和公共机构采用NEM区块链。如果你能够取得政府或公共机构合同，使用NEM区块链构建解决方案，你将获得等值5,000美元的XEM的奖金。

      根据以下条款和条件，前10名可获得赏金:

      1. 合同必须与国家或州政府或与其相关的机构
      2. 合同价值必须至少为50,000美元
      3. 至少在该国的一个主要媒体平台上进行报道
      4. 一旦提供以下资料，奖金将被支付：
        • 合同的复件
        • 收到合同价值至少10％的付款证明
        • 证明工作已经开始履行合同
      5. 每个国家只分配1次奖金
      6. 赏金小组有最终解释权
